My browser isn't supported? What should I know?

Some browsers don't have all the functionality other browsers have. The Pomodoro timer is heavily reliant on Chromium/WebKit based browsers because they have every feature the timer needs. In the future our small dev team will try to make workarounds for this very reason, but currently you will have to use one of the below browsers to get full functionality.

  • Google Chrome
  • Microsoft Edge
  • Opera
  • Opera GX
  • Brave
  • Vivaldi
  • Yandex Browser
  • Chromium
  • Safari
  • Any other up to date Chromium or WebKit based browser

If you don't need full functionality, you can use the below browsers

  • Firefox
  • Pale Moon
  • Any other browser that isn't Chromium or WebKit based

Wondering what you're missing? Refer to this table.

Browser Feature Engine Explanation
Firefox Wake Lock Gecko Firefox doesn't support wake lock, which prevents the computer from sleeping or the display from turning off.
Pale Moon Form Submission, Wake Lock Goanna Pale Moon doesn't support wake lock, which prevents the computer from sleeping or the display from turning off. Pale Moon also doesn't support event.preventDefault() on a form, meaning the page will reload when you try to start the timer.

You are using an unknown browser engine.

Note: if you are on mobile, some features may not be supported. Mobile browsers are not tested for the Pomodoro timer.
